# deepgram_project (Data Source)

Looks up a single Deepgram project by name and returns its details.

Returns an error if no project matches the name, or if multiple projects share the same name (use `deepgram_projects` to list all and select manually in that case).

## Example Usage

### Look up a project and create a key in it

```hcl
data "deepgram_project" "main" {
  name = "My Production Project"
}

resource "deepgram_key" "app" {
  project_id = data.deepgram_project.main.project_id
  comment    = "App key"
  scopes     = toset(["usage:read", "usage:write"])
}
```

### Output the project ID

```hcl
data "deepgram_project" "main" {
  name = "My Production Project"
}

output "project_id" {
  value = data.deepgram_project.main.project_id
}
```
