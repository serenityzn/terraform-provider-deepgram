# deepgram_projects (Data Source)

Retrieves all Deepgram projects accessible with the configured API key.

## Example Usage

### List all projects

```hcl
data "deepgram_projects" "all" {}

output "project_ids" {
  value = [for p in data.deepgram_projects.all.projects : p.project_id]
}

output "project_names" {
  value = [for p in data.deepgram_projects.all.projects : p.name]
}
```

### Use a project ID in another resource

```hcl
data "deepgram_projects" "all" {}

resource "deepgram_key" "app" {
  project_id = data.deepgram_projects.all.projects[0].project_id
  comment    = "App key"
  scopes     = toset(["usage:read"])
}
```
